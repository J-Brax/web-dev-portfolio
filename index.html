>> About Me

Get to know me

Web development clicked for me during third year Computer Engineering in a Software Design course. I loved that I could build anything, so I taught myself the practical side beyond what school offered.

I did a 240-hour college internship at Cauld and Clark IT Solutions maintaining inventory systems and building an IT ticketing platform, then joined proper internship for three months after graduating in 2025. But I realized I was stacking frameworks without solid fundamentals, so I stepped back to join Uplift Code Camp to master the basics properly.

I've built with React, Next.js, TypeScript, Tailwind, PostgreSQL, and MongoDB—including a school management system for my program chair—but now I'm rebuilding from HTML, CSS, and JavaScript up to make sure every layer is solid.
